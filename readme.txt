== Description ==

= Referboard Retailer WooCommerce Plug-in =

Referboard is an exciting new platform which enables you to generate more website traffic and online sales, by enabling your audience to refer your products to their friends and followers. This Plug-in enables your website to easily integrate with the Referboard platform and get your products online quicker.

== Installation ==

In order to complete the Referboard Retailer plug-in installation, you must be registered with Referboard. If you haven't registered your eCommerce website with Referboard yet, you can do it quickly <a href="https://www.referboard.com/registerretail">here</a>.

= Instructions: =
1. Download the Referboard Retailer plugin <a href="https://wordpress.org/plugins/referboard-woocommerce/">here</a>
2. Log in to your Referboard account, select 'Integrate Code' tab in the left column. Copy the API key.
3. Log in to your WooCommerce website settings with admin privileges and activate the Referboard Retailer plugin. Select the 'ReferboardButton' menu after activation, enter the API key and type the currency "AUD" and press Save.
4. Return to your Referboard account, go to the 'Test' tab in the left column and complete the test process to activate your account.

If you require any additional help please <a href="https://www.referboard.com/site/index?action=login">log in</a> to your Referboard Retailer account and activate the live chat platform in the bottom right corner of the screen. Alternatively, you can contact support@referboard.com.
== Frequently Asked Questions ==

= Where can I find my API key? =

Log in (https://www.referboard.com/site/index?action=login) to your Referboard Retailer account and go to the 'Integrate Code’ tab in the left panel. Your API key will display at the top of this tabs content.

= How can I change the style of the Refer button? =

Log in (https://www.referboard.com/site/index?action=login) to your Referboard Retailer account and go to the 'Customise’ tab in the left panel. Select one of the Refer button options to display on your website.
